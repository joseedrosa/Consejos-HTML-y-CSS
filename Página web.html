    Introducción a HTML


    Premisas básicas


    Primero escribimos la dirección o URL del sitio web al que queremos entrar, a continuación y sin que nosotros nos demos cuenta, nuestro navegador solicita la página web al servidor que la alberga. Acto seguido, el servidor envía de vuelta los datos a nuestro ordenador a través de Internet. Finalmente, nuestro navegador interpretará los datos, mostrando el resultado en la pantalla de nuestro ordenador.



    Entrando más en detalle:


    • -Nuestro ordenador de casa envía una petición al servidor y le devuelve una respuesta con los datos en lenguaje html.

    • -A partir de aquí, el navegador que estemos utilizando interpretará esos datos y los mostrará en la pantalla.

    • -Del mismo modo podemos obtener visualizaciones distintas para cada navegador debido a que son éstos los que interpretan los datos obtenidos.

    • -Por ejemplo, podemos usar como navegador Internet Explorer o Firefox, y según usemos uno u otro obtener resultados distintos porque cada navegador interpreta la información de una manera distinta.

      ¿Qué es HTML y HTTP?


      • -HTTP: servidor y cliente:

        • El http son las siglas de “Hypertext Transfer Protocol” es un protocolo de transferencia donde se utiliza un sistema mediante el cual se permite la transferencia de información entre diferentes servicios y los clientes que utilizan páginas web. Get Sent es una solicitud, que funciona escribiendo la palabra “Get” seguido del nombre de documento que quieras, que un usuario pide a un servidor para poder acceder a la página que desee y este le facilitará todo su código HTML.


      • -HTML:

        • Es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la sigla que corresponde a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto, que podría ser traducido como Lenguaje de Formato de Documentos para Hipertexto. Se encarga de desarrollar una descripción sobre los contenidos que aparecen como textos y sobre su estructura, complementando dicho texto con diversos objetos (como fotografías, animaciones, etc).


        Estructura básica de HTML


        Sintaxis básica


        • -Elemento:

          • Es el componente básico de una página web, una página web está compuesta por un conjunto de elementos escritos en un orden concreto. Un elemento se compone de una etiqueta inicial o de apertura, y una etiqueta final o de cierre. Hay más de 100 etiquetas, aqui te mostramos algunos ejemplos:


        Etiqueta Descripción
        < html>...< /html> Declara que la página web está escrita en HTML.
        < head>...< /head> Delimita el encabezado de la página.
        < title>...< /title> Define el título (no mostrado en la página).
        < body>...< /body> Delimita el cuerpo de la página.
        < p>...< /p> Empieza un párrafo.


        • -Atributo:

          • La etiqueta inicial puede llevar atributos, pero la final nunca lleva. En HTML, los atributos pueden llevar un valor.


        • -Contenido:

          • Es todo lo que se escriba entre la etiqueta inicial y final: el contenido puede estar formado por otras etiquetas de HTML o puede ser simplemente texto como en este ejemplo.


        • -La etiqueta < html>:

          • Es la etiqueta principal, sólo puede existir una y toda la página web debe escribirse entre la etiqueta inicial y final de html. La etiqueta head define la primera parte de una página web, la cabecera. Normalmente lo que se escribe en la cabecera no aparece representado en la página web cuando se visualiza en un navegador web, sino que se emplea para definir los metadatos, información sobre la página o instrucciones sobre cómo procesar la página web. Dentro de la cabecera se puede escribir la etiqueta title que es obligatoria, sólo puede haber una, y se emplea para definir el título de la página. La etiqueta body define la segunda parte de una página web, el cuerpo. Lo que se escribe en el cuerpo sí que aparece representado en la página web cuando se visualiza en un navegador web.


        • -< h1>, < h2> y hasta < h6>:

          • Se emplean para definir los encabezados o títulos en la página. Existen seis niveles de importancia, siendo < h1> el más importante, y < h6> el nivel menos importante. Normalmente, usarás los niveles < h1>, < h2> y < h3>. p se emplea para definir un párrafo de texto. < strong>, se emplea para destacar un fragmento de texto importante. < em> se emplea para definir un texto enfatizado.


        Etiquetas básicas


        Vamos a explicar conceptos básicos de HTML. En primer lugar veremos cuál es la estructura básica que toda página HTML debe cumplir. Para ello introduciremos las etiquetas HTML, HEAD y BODY, y sus respectivas etiquetas de cierre. Toda página web viene definida con la siguiente estructura básica:



        HTML


        La etiqueta < html> define que se trata de código HTML.


        HEAD


        La cabecera del documento es la sección comprendida entre < head> y < /head>. En ella se debe encontrar, obligatoriamente, el título (entre las etiquetas < title> y < /title>). El título de la página debe describir su contenido Otras etiquetas pueden ir incluidas en < Head> (son opcionales) y son las siguientes:


        • -< Style>:

          • Incluye estilos de CSS que permiten dotar de colores, bordes, imágenes de fondo, etc. a los elementos de la página web.


        • -< Meta>:

          • Sirven para incluir información que no se muestra como parte de la página web pero sirve para informar de características de la página web, como su descripción breve y sus palabras clave.


        • -< Link>:

          • La etiqueta < link>, permite realizar la carga y establecer la relación existente entre el documento web actual y un recurso externo. Esta etiqueta es normalmente muy utilizada para realizar la carga de hojas de estilo para el documento web. Entre los atributos a destacar que tiene la etiqueta se encuentran: href (permite establecer URLs), hreflang (establece el lenguaje del documento), size (establece el tamaño del atributo) media (sirve para indicar el tipo de dispositivo para el cual se esta realizando el enlace.) entre otras.


          BODY


          El cuerpo (body) del documento html es normalmente lo más importante. Es aquí donde debemos colocar el contenido de nuestra página: texto, fotos, etc. El cuerpo está delimitado por las etiquetas < body> y < /body>.

          Introducción a CSS


          Las Hojas de Estilo en Cascada (Cascading Style Sheets) definen el formato o estilo para la presentación del documento.


          Inclusión en el documento HTML


          Las hojas de estilo se pueden incluir de las siguientes maneras:

          Inline:


          Dentro de la misma línea de código por lo que solo la afecta a ella. En etiquetas mediante el atributo global style como en el ejemplo: < p style="color red">Texto afectado< /p>

          Internal:


          Dentro del HTML, afectándolo todo. En la cabecera del documento mediante la etiqueta < style>: < head> < style> hr{color red} < /style> < /head>

          External

          En un fichero independiente. < head> < link rel="stylesheet" href="css/fichero.css"/> < /head>

          Selectores


          Los selectores son las principales etiquetas de CSS. Los principales son:

          Elemento:


          Define el estilo para ese tipo de etiqueta. Ejemplo: body{color:black}


          Clases (class):


          Estilo para las etiquetas cuyo atributo class tome ese valor. Ejemplo: En CSS: .centrado{text-align:center} En HTML: < p class="centrado">...< /p>


          Identificador (id):


          Estilo a aplicar al elemento con ese identificador (atributo id). Ejemplo: En CSS: #rojo{color:red} En HTML: < p id="rojo">...< /p>


          Modelo de caja



          Margen (margin):


          Distancia que queremos que haya entre nuestro elemento y el resto de los elementos. Es invisible. Si queremos que cada uno de los márgenes laterales sea diferente utilizamos: -margin-top:10px -margin-bottom:20px -margin-left:30px -margin-right:40px Otra opción es escribir los cuatro valores en sentido horario por lo que los valores anteriores quedarían así: margin:10px 40px 20px 30px Para centrar el contenido de manera horizontal utilizamos margin:auto


          Borde (border):


          Línea que delimita nuestra caja. Podemos definir su color, grosor y modelo (están predeterminados como el "solid"). Si queremos que cada uno de sus lados tenga un grosor diferente podemos diferenciarlos utilizando "border-top", "border-bottom", "border-left" y "border right". border:20px solid rgb(23,35,67)


          Relleno (padding):


          Distancia entre el borde y la letra: padding:20px


          Contenido:


          La información.


          Posicionamiento


          Utilizando el modelo de caja, podemos colocar la información en la posición que más nos convenga. Estos son los tipos de posicionamiento de los que disponemos:


          Posicionamiento fijo


          Independientemente del tipo de posicionamiento fijo, disponemos de estos cuantro valores:

          • -top:valor
          • -bottom:valor
          • -right:valor
          • -left:valor

          NOTA:no es necesario utilizarlos todos, solo los que nos hagan falta para colocar la caja donde queramos.
          • -Relativa(position:relative):Posicionamos el elemento con respecto al flujo normal de la página; es decir, con respecto al lugar que por defecto le correspondería.

          • -Absoluta(position:absolute):Posicionamos el elemento tomando como referencia la ventana del navegador o el elemento posicionado (que tenga cualquier valor de position) más cercano si es padre del elemento que queremos posicionar.

          Posicionamiento flotante


          • Float:

            • Aplicable a left y right, hace que el elemento pierda el flujo normal de ubicación y se coloque a la derecha o a la izquierda del contenedor, ocupando el resto de elementos del espacio. Es decir, el elemento con float se colocará por encima de todo lo demás. float:left


          • Clear:

            • Aplicable a izquierda, derecha o ambos, impide que un elemento flote, a izquierda o derecha, sobre él. clear:both
